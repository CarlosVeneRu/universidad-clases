# Sistema de Gestión de Clases Universitarias

Sistema web para administrar las clases impartidas en la Universidad. Permite cargar masivamente desde Excel, buscar materias/maestros/salones, detectar choques de horarios y generar reportes.

## Funcionalidades principales

- Carga automática desde archivos Excel
- Búsqueda y filtrado de materias, maestros y salones
- Detección automática de choques de horarios
- Reportes de uso de salones y carga docente
- Archivado automático de clases vencidas

## Tecnologías

- **Backend / DB**: Supabase (PostgreSQL)
- **Frontend**: Streamlit
- **Lenguaje**: Python 3.11+
